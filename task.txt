Создать приложение которое будет отображать список курсов.
1) создать компонент *<app-logo>* - в который поместить любую картинку в качестве логотипа
2) создать компонент *<app-header>* - в котором будет логотип + название приложения
3) создать компонент *<course-item>* который принимает и отображает данные для конкретного курса. Если у курса поле *topRated* true - раскрасить кружок в желтый цвет. Так же компонент должен реагировать на нажате по тайтлу - прокидывать данные (тайтл) на верх (паренту).
4) создать компонент *<course-list>* котрый содержит в себе список всех курсов и отображает <course-item> в цикле. написать метод `getTitle()` - который выводит в консоль тайтл выбранного курса.
5) отобразить *<app-header>* *<course-list>* на странице.

* Создать интерфейс CourseModel.
* Все свойства должны быть типизированны.

** по нажатию на кружок - изменять поле topRated.
