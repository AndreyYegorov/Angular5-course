ООП задание

1) Создать абстрактный класс *Article* у которого есть:
    - свойство `createdAt` - дата создания
    - публичный метод `getCreatedDate()` - возвращает дату создания
    - публичный метод `getAuthor()` - возвращает имя автора
    - абстрактный метод `getTitle()` - возвращает название.
  Свойство `createdAt` должно автоматически устанавливатся при созданнии экземпляра класса.

2) Создать 2 класса *Book* & *Magazine* которые наследуются от *Article*.
    в классе *Book* реализовать метод `getPublisher()` - который вернет имя паблишера
        - метод `getPublisher` описать через интерфейс
    в классе *Magazine* реализовать метод `getAgency()` - который вернет имя агенства
        - метод `getAgency` описать через интерфейс

Создать интерфейсы *BookModel* & *MagazineModel*.
Все свойства и методы должны быть типизированными.
